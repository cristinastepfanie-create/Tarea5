Tutorial: Dominando los Botones

1. ¿Cómo crear un botón básico?

La forma más moderna y recomendada es usar la etiqueta <button>. A diferencia de los antiguos inputs, esta etiqueta te permite incluir texto, iconos o imágenes dentro.

Ejemplo de código:

<button>Haz Clic</button>

Es importante siempre definir el atributo type para saber qué hace el botón.



2. Los tipos de botones (Attribute Type)

Existen tres valores principales para el atributo type que debes conocer:

  • type="submit": Envía formularios. Es el valor por defecto.
  • type="reset": Borra el contenido de un formulario.
  • type="button": No hace nada por sí solo. Se usa con JavaScript (como el botón de subir que usaremos al final).



3. Dando estilos con CSS (Básico)

Los botones por defecto se ven grises y aburridos. Para arreglarlos, usamos CSS. Las propiedades más comunes son:

background-color: Cambia el color de fondo.

color: Cambia el color del texto.

padding: Agrega espacio interno para que el botón se vea más grande.

border-radius: Redondea las esquinas.




4. Estados del botón (:hover y :active)

Para que el botón se sienta interactivo, usamos "pseudo-clases" en CSS:

button:hover -> Define cómo se ve el botón cuando pasas el mouse por encima (ej. cambiar el color un poco más oscuro).

button:active -> Define cómo se ve cuando lo estás presionando (ej. moverlo un poco hacia abajo).




5. Accesibilidad

Nunca olvides la accesibilidad. Asegúrate de que el contraste entre el texto y el fondo sea suficiente para que personas con problemas de visión puedan leerlo.

También usa el atributo aria-label si el botón solo tiene un icono y no texto, para que los lectores de pantalla sepan qué hace.
